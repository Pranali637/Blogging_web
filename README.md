# Blogging_web

This project is a backend-focused blogging platform built to understand how real-world web applications work without relying on frontend frameworks.

Instead of React or SPA tools, this app uses server-side rendering with EJS, making it simple, fast, and beginner-friendly.

Built while learning from tutorials.

Why this project exists
Most beginners jump straight to React.

This project does the opposite.

It focuses on:

How servers handle requests
How authentication actually works
How data flows from MongoDB → Express → EJS
How cookies & JWTs manage login sessions
Tech Stack (What I Actually Used)
Core

Node.js
Express.js
Database

MongoDB
Mongoose ODM
Rendering

EJS (Server-side templates)
Auth & Security

JWT (stored in cookies)
Custom authentication middleware
Utilities
